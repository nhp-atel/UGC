Project Structure & Data Flow

How files are organized and how data moves through the system

Folder Structure

  ├── worker.ts                  ← Temporal worker entry

  │
  ├── workflows/
  │     ├── generateQuoteContentWorkflow.ts

  │     └── generateQuoteContentTypes.ts
  │
  ├── activities/
  │     └── directorActivity.ts  ← single activity
  │           calls DirectorAgent.run()
  │
  ├── agents/
  │     ├── base/
  │     │     ├── BaseAgent.ts   ← run/evaluate loop
  │     │     └── types.ts       ← AgentResult, Evaluation

  │     ├── DirectorAgent.ts     ← LLM orchestrator
  │     ├── ResearchAgent.ts     ← quote research
  │     ├── CreativeBriefAgent.ts← creative direction

  ├── tools/

  │     ├── videoService.ts      ← [mocked] video gen

        └── config.ts            ← env vars, constants
Responsibility Map
workflows/
  Temporal workflow definition
  Signals, queries, state
  Calls directorActivity
  Handles approval flow

activities/
  Single activity: runDirector
  Bridges Temporal ↔ Agent layer
  Handles activity-level timeout

agents/
  AI decision-making logic
  BaseAgent: run → evaluate → retry
  DirectorAgent: tool-calling loop
  Sub-agents: domain-specific AI

tools/
  OpenAI function definitions
  Bridges Director ↔ Sub-agents
  Bridges Director ↔ Services
  Tool registry for Director

services/
  External API integrations
  No AI logic — pure I/O
  Mocked now, real later

lib/
  Shared infrastructure
  OpenAI client, config

Data Flow — A Single Workflow Execution

End-to-End Data Flow
startWorkflow.ts
  │  { topic: "discipline", tone: "cinematic", platform: "youtube_shorts", ... }
  ▼
generateQuoteContentWorkflow.ts
  │  state.stage = "RUNNING_DIRECTOR"
  │  calls directorActivity(input)
  ▼
directorActivity.ts
  │  creates DirectorAgent
  │  calls director.run(input)
  ▼
DirectorAgent.ts ──── OpenAI chat completion loop ────────────────────────
  │                                                                       │
  │  Turn 1: "I need quotes about discipline"                             │
  │  → tool_call: research_quotes({topic, tone, count: 5})                │
  │     │                                                                 │
  │     ▼  researchTool.ts → ResearchAgent.run()                          │
  │        │  attempt 1: generate quotes → evaluate (8/10 ✓)              │
  │        └─ returns: { candidates: [...], selected: {text, author} }    │
  │                                                                       │
  │  Turn 2: Director evaluates: "Good quote. Now creative brief."        │
  │  → tool_call: build_creative_brief({quote, tone, platform})           │
  │     │                                                                 │
  │     ▼  creativeTool.ts → CreativeBriefAgent.run()                     │
  │        │  attempt 1: generate brief → evaluate (6/10 ✗)               │
  │        │  attempt 2: retry with feedback → evaluate (8/10 ✓)          │
  │        └─ returns: { hook, visualConcept, voiceoverText, audioMood }  │
  │                                                                       │
  │  Turn 3: Director evaluates: "Hook is good. Write the post."          │
  │  → tool_call: write_post({quote, brief, hashtags})                    │
  │     │                                                                 │
  │     ▼  postTool.ts → PostAgent.run()                                  │
  │        │  attempt 1: generate post → evaluate (9/10 ✓)                │
  │        └─ returns: { title, description, hashtags }                   │
  │                                                                       │
  │  Turn 4: Director evaluates: "Everything cohesive. Generate video."   │
  │  → tool_call: generate_video({quote, brief})                          │
  │     └─ videoTool.ts → videoService (mocked paths)                     │
  │                                                                       │
  │  Turn 5: "Save the draft."                                            │
  │  → tool_call: save_draft({...all content...})                         │
  │     └─ saveTool.ts → storageService (mocked)                          │
  │                                                                       │
  │  Turn 6: "All done." → returns final result                           │
  ├───────────────────────────────────────────────────────────────────────┘
  ▼
Back to workflow
  │  state = result from Director
  │  if requireApproval → AWAITING_APPROVAL (signals)
  │  if auto_publish → calls publishActivity
  │  return GenerateQuoteContentResult
  ▼
startWorkflow.ts
  │  logs final JSON result
  ▼
Done.
