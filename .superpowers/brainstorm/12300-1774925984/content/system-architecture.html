Key Design Decisions

Director = Activity

The Director Agent runs inside a single Temporal activity. Temporal handles durability/retries at the workflow level. The Director handles intelligence inside.

Tool-Based Sub-Agents

Each sub-agent is exposed to the Director as a tool. The Director calls tools via OpenAI function calling. Sub-agents have their own internal quality loops.

Double Quality Layer

Layer 1: Each agent self-evaluates (try → check → retry). Layer 2: Director evaluates the result and can re-invoke with feedback.

Temporal Stays Simple

The workflow only does: call Director → handle approval signals → return result. All intelligence lives in the agent layer, not the workflow.
